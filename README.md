# SmartLearn AI

SmartLearn AI is an AI-powered learning assistant that creates personalized study materials for students.

## Features

- Personalized explanations
- Quiz generation
- Flashcards
- Revision notes
- 30-minute study plans
- Knowledge gap analysis
- Copy notes feature
- Subject and difficulty customization

## Tech Stack

- Python
- Flask
- Gemini API
- HTML
- CSS
- JavaScript
- Markdown

## How It Works

1. Enter a topic.
2. Select subject.
3. Select difficulty.
4. Select learning goal.
5. SmartLearn AI generates:
   - Explanation
   - Key Points
   - Quiz
   - Flashcards
   - Study Plan
   - Knowledge Gap Analysis
   - Revision Notes
   - Summary
